xenji.com PEAR channel

Registering the channel: 
pear channel-discover pear.xenji.com
Listing available packages: 
pear remote-list -c xenji
Installing a package: 
pear install xenji/package_name
Installing a specific version/stability: 
pear install xenji/package_name-1.0.0
pear install xenji/package_name-beta
Receiving updates via a feed: 
http://pear.xenji.com/feed.xml
